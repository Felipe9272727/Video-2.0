# RIFT//RIDER

Um endless runner 3D mobile feito com Three.js. Pilote uma nave através de um rasgo cósmico procedural, colete cristais, encadeie combos e use o impulso para sobreviver às zonas do abismo.

## Recursos

- Controles mobile por joystick e botão de boost
- Suporte a teclado (WASD/setas + espaço)
- Obstáculos e ondas procedurais
- Cinco zonas visuais progressivas
- Bloom, partículas, câmera dinâmica e efeitos de velocidade
- Áudio sintetizado em tempo real com Web Audio
- Escudo, energia, combo, quase-acidentes e recorde local
- Interface responsiva em português

## Rodar localmente

```bash
npm install
npm run dev
```

Para gerar a versão de produção:

```bash
npm run build
```

## Controles

- **Mobile:** arraste o joystick para pilotar e segure **BOOST** para acelerar.
- **Desktop:** use WASD ou setas; segure espaço para o boost.

Recomendado jogar no celular em modo paisagem e com som.
